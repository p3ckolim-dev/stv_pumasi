# pumasi (품앗이)

`pumasi`(`pms`, 한국어 이름 `품앗이`)는 Stardew Valley용 SMAPI 모드 프로토타입입니다. 기준 언어는 한국어이며, 영어 문서는 보조 번역으로 함께 제공합니다. 이 모드는 호스트 권한 농장 도우미로서 Stardew Valley 질문에 답하고, 공유 할 일 목록을 관리하며, 안전하게 확인된 반복 농장일을 실행하는 것을 목표로 합니다.

## 이름의 유래

`pumasi`는 한국 전통 농촌 문화인 `품앗이`에서 따온 이름입니다. 품앗이는 농번기처럼 한 집의 일손만으로 감당하기 어려운 일이 있을 때 이웃이나 친인척이 노동을 보태고, 이후 비슷한 방식으로 그 도움을 되갚는 상호부조 관습입니다.

이 모드의 도우미도 같은 의미를 이어받아, 플레이어나 멀티플레이 그룹이 반복적인 농장일에 시간이 부족할 때 함께 일을 나누는 존재를 목표로 합니다.

| 항목 | 내용 |
| --- | --- |
| 현재 버전 | `0.1.22` |
| 기준 언어 | 한국어 |
| 게임 / 모드 로더 | Stardew Valley 1.6, SMAPI 4.x |
| 주요 명령어 | `/pms`, `/pms ask`, `/pms scan`, `/pms todo`, `/pms animals on|off` |
| AI 제공자 | Gemini REST API |
| 멀티플레이 모델 | 호스트 권한 중심 |
| 상태 | MVP/프로토타입. 아직 바닐라식 NPC 시스템은 아닙니다. |

## 개요

Pumasi는 반복적인 농장일을 함께 나누는 협동형 도우미로 설계되었습니다. 실제 멀티플레이 농장 일꾼을 새로 만드는 방식은 아니며, AI 호출, 작업 큐 변경, 월드 상태를 바꾸는 행동은 모두 호스트가 소유합니다. 게스트에게는 정리된 도우미 상태와 할 일 목록만 동기화됩니다.

도우미는 Stardew Valley 관련 질문에 답하고, 저장 파일별 최근 대화 맥락을 유지하며, 호스트가 스캔한 안전한 농장 작업을 공유 할 일 목록으로 바꿀 수 있습니다.

## 현재 기능

| 영역 | 현재 동작 |
| --- | --- |
| 싱글플레이 | 로컬 플레이어가 호스트이므로 명령어와 안전 작업 실행이 로컬에서 동작합니다. |
| 멀티플레이 | 게스트는 도우미 요청을 보낼 수 있고, AI 호출과 월드 변경은 호스트가 수행합니다. |
| 할 일 보드 | HUD에 작은 Pumasi 아이콘이 표시되며, 클릭하면 할 일 보드 팝업이 열립니다. |
| 할 일 행 | 보드와 `/pms todo`에서 상태, 우선순위, 출처, 대상 타일, 완료/건너뜀/실패 이유를 읽을 수 있게 표시합니다. |
| 설정 UI | Stardew Valley 게임 메뉴의 Pumasi 탭에서 스크롤 가능한 빠른 설정과 한국어/영어 UI 언어 선택을 제공합니다. |
| AI 계획 | Gemini가 작업 계획에 사용되지만, 호스트가 스캔한 안전 후보 작업과 일치하는 항목만 큐에 들어갑니다. |
| 위키 답변 | Stardew Valley 정보 답변은 한국어 Stardew Valley Wiki를 기준으로 하며, 검색어는 조회 전에 정규화됩니다. |
| 대화 맥락 | 저장 파일별 최근 사용자/도우미 대화를 유지해 `그거`, `방금 말한 거`, `do that` 같은 후속 표현을 맥락에 맞게 처리합니다. |

## 지원하는 작업 실행

| 분류 | 지원 작업 |
| --- | --- |
| 작물 | 마른 작물 물주기, 수확 가능한 작물 수확, 스프링클러 근처 빈 땅 갈기. |
| 기계 | 완료된 기계 생산물 수거. |
| 동물 | 동물 건물 건초 보충, 아직 쓰다듬지 않은 동물 쓰다듬기, 바닥에 떨어진 동물 생산물 수거. |

바닥에 떨어진 동물 생산물은 가능한 경우 같은 아이템이 들어 있는 상자에 보관하고, 그렇지 않으면 호스트 인벤토리에 넣습니다. 둘 다 받을 수 없으면 Pumasi는 아이템을 제자리에 남겨 둡니다.

## 안전 모델

| 안전 경계 | 규칙 |
| --- | --- |
| 호스트 권한 | 게스트는 월드 상태를 바꾸는 자동화를 로컬에서 실행하지 않습니다. |
| AI 계획 | Gemini는 스캔된 후보 작업 중에서만 선택할 수 있으며, 호스트는 스캔되지 않은 작업 키를 거부합니다. |
| 동물 작업 | 동물 관련 작업은 `Animals` 작업 카테고리가 켜져 있을 때만 실행됩니다. |
| 생산물 보관 | 바닥의 동물 생산물은 같은 아이템 상자를 먼저 사용하고, 임의의 빈 상자는 사용하지 않습니다. |
| 보상 / 경험치 | Pumasi는 경험치나 보상을 수동으로 지급하거나 게스트에게 나누지 않습니다. 호스트 측 바닐라 효과는 호스트에게 귀속됩니다. |
| 비밀값 | Gemini API 키는 로컬에만 남으며 멀티플레이 동기화 메시지에 포함되지 않습니다. |

## 버전 및 릴리스 노트

- 현재 모드 버전: `0.1.22`.
- SMAPI `UpdateKeys`는 새 릴리스를 플레이어에게 알려 주지만, 새 zip 설치는 여전히 수동입니다.
- 빌드 결과물 패키지 이름은 `Pumasi 0.1.22.zip`입니다.
